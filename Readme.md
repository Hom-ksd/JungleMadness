# Jungle Madness

Load `JungleMadness.zip` then unzip file run the JungleMadness.exe for play a Game and enjoy!

## How to play 
**'Fight All Monster'**
  
| Key / Input  | Action |
| ------------- | ------------- |
| A D | Move  |
| J or Left Click  | Attack |
| K or Right Click | Skill |
| F | Interact |
| P | Pause |
| Space Bar | Jump |
| Shift | Roll |
